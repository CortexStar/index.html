To enable the one-touch workflow, you must disable the browser's "Save As" dialog:

  1. Go to your browser's Settings and search for "Downloads".
  2. Find the option labeled "Ask where to save each file before downloading".
  3. Turn this setting OFF.
  4. You can also set a default download location (e.g., your Desktop or Downloads folder).
